# Week 1: Introduction to Cloud Computing
------------------------------
Day 1: What is Cloud Computing?
What I Learned:
Cloud computing is renting computer power and storage from big companies insted of buying your own servers.

SIMPLE EXPLANATION:
. Without Cloud (Old Ways):
 . Your Company buys expensive servers
 . Stores them in an office 
 . Pays for electricity, Cooling, Respairs etc..
 . If server breaks = Problem 
 . Very Expensive 

. With Cloud(New Ways):
 . Rent Server from AWS, Azure, or Google
 . Pay monthly only for what you use 
 . Company handles repairs and backups
 . Can grow or shrink whenever you want
 . Much cheaper and safer

KEY POINT:
 Cloud = Just computers in big data centers that you rent instead of own System.

 In Cloud Computing i Learnt that we have two main cloud model.

1. Service Model( How you use the Cloud)
2. Deployment Model( how the cloud is set up)

What is Service Model and Deployment Model:
 1. Service Model describes what part of the cloud service we use or control
 2. Deployment Model describes where the cloud is hosted and who can access it 

 KEY POINT:
 Service Model = What you use
 Deployment Model = Where it runs and who can access
